# Ansella Learning Platform — LMS + E-Learning + E-Commerce

Production-grade LMS platform for Blockchain, Cryptocurrencies, and Artificial Intelligence.

## Features & Architecture
- **Role-based Dashboards**: Admin, Instructor, Student
- **Kajabi-Style Community**: Topics, discussions, resource sharing, upvotes, and instructor/affiliate leaderboard
- **Flexible Billing**: Monthly & Annual plans with 10% discount
- **Multi-channel Payments**: Mobile Money (PawaPay), Cards (Moko), PayPal
- **Automated Certification & Grading**: PDF & QR verification, hybrid scoring

## Tech Stack
- Next.js 14 / 16 (App Router & Turbopack)
- React, TypeScript, Tailwind CSS, shadcn/ui
- Supabase PostgreSQL, Auth.js

## Deploy on Vercel
Connected to Vercel Production deployment pipeline.
